## 🔍 调试技巧

### 查看完整日志
```powershell
# Windows
type C:\Users\%USERNAME%\AppData\Roaming\tft-hextech-helper\logs\tft-2026-03-03.log

# 或用记事本打开
notepad C:\Users\%USERNAME%\AppData\Roaming\tft-hextech-helper\logs\tft-2026-03-03.log
```

### 搜索关键信息
在日志文件中搜索：
- `阶段确认成功` - 确认识别是否成功
- `阶段不合理` - 查找不合理的识别
- `收到取消信号` - 确认F1是否响应
- `ERROR` - 查找所有错误

### 常见问题排查

**问题1：日志文件还是不存在**
- 检查控制台是否有 `[Logger] 初始化文件日志失败` 错误
- 确认路径权限（Windows需要对 AppData 有写权限）

**问题2：F1还是停不下来**
- 检查是否在 LobbyState 的 while 循环中卡住
- 日志搜索 `正在退出房间`，看重试次数是否过多

**问题3：阶段识别还是错误**
- 日志搜索 `OCR原始`，查看原始识别结果
- 检查是否有 `阶段识别结果不合理` 的警告
- 尝试调整模拟器窗口大小到 1024×768

**问题4：还是只走位**
- 确认阶段识别成功（必须看到 `✅ 阶段确认成功`）
- 检查是否选择了阵容（设置页面）
- 查找日志中是否有策略执行相关的信息
